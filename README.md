# Snake Game

A retro Snake Game inspired by old Nokia mobile phones, built using Java Swing.

## Features

* Red snake with eyes
* White food balls
* Large bonus red ball every 10th food
* Score tracking
* Snake growth mechanics
* Start and restart options
* Self-collision detection
* Wall collision detection
* Fixed-size game window

## Technologies Used

* Java
* Java Swing

## How to Run

```bash
javac SnakeGame.java
java SnakeGame
```
